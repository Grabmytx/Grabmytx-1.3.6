Dans cette démonstration,
Nous ne téléchargerons pas le "logiciel" depuis le site web
pour un gain de temps et pour éviter tout problème
de compatibilité avec le navigateur

Problème de compatibilité avec le navigateur :

L'attribut download indique la cible (le fichier spécifié dans l'attribut href) qui sera téléchargée.
Cela évite que lorsque l’utilisateur clique sur l'hyperlien, le fichier s’ouvre juste en mode lecture dans le navigateur.
Mais les versions trop anciennes des navigateurs ne prennent pas en charge cette attribut (voir image ci-dessous).
